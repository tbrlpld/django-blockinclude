Basic include

Below is a paragraph included from a another template. This is the plain include functionality with the default include tag.

{% include 'includes/static-paragraph.html' %}

You can also pass variables to included. Here is a box that receives a string as the content variable.

{% include 'includes/the-box.html' with content="Lorem ipsum dolor sit amet, consectetur adipiscing elit." %}

You can even pass markup as variable. (Notice the bolded "Lorem").

{% include 'includes/the-box.html' with content="Lorem ipsum dolor sit amet, consectetur adipiscing elit." %}

Unfortunately, you can only pass single lines. When you break the tag, then the tag does not work anymore and is printed in the output.
