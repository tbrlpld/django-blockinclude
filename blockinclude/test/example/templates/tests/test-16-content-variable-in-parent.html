{% extends '_test-base.html' %} {% load blockinclude %} {% block content %}

When the parent template defines a content variable, this is overridden by the content of the blockinclude when the template is included. In the parent itself, the variable is not affected by that.

{% with content="Adipisci atque aut incidunt" %} {% blockinclude 'includes/the-slotted-box.html' %} Lorem ipsum dolor sit amet, consectetur adipiscing elit. {% endblockinclude %}
{{ content }}
{% endwith %}
{% endblock %}
