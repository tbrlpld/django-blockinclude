Block include renders output from parent

With the new blockinclude tag, we can pass a block of rendered content directly to the included template. The block is rendered in the context of the parent and then passed to the included template. The rendered markup is available as the content variable in the included template.

{% blockinclude 'includes/the-box.html' %} Lorem ipsum dolor sit amet, consectetur adipiscing elit. {% endblockinclude %}
{% endblock %}
