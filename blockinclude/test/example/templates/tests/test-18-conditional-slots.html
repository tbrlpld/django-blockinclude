When the slot is nested inside of a if block or similar, then it is not the child of the blockinclude anymore. In those cases, it is not rendered, and the variables are not available in the included template. If you need to conditionally change the content of a slot, do that inside of the slot.

{% blockinclude 'includes/the-slotted-box.html' %} {% if False %} {% slot header %} Phasellus nulla justo, dictum eu diam non, congue consequat ipsum. {% endslot %} {% endif %} Lorem ipsum dolor sit amet, consectetur adipiscing elit. {% if True %} {% slot footer %} Minima modi non quia. {% endslot %} {% endif %} {% endblockinclude %}
{% endblock %}
